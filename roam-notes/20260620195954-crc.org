#+title: CRC
* 题目：CRC 循环冗余校验计算

在计算机网络的数据链路层中，某发送端需要使用 CRC 方法对一段二进制数据进行差错检测。

已知待发送的数据比特串为：

#+begin_src text
D = 1101011011
#+end_src

生成多项式为：

#+begin_src text
G(x) = x^4 + x + 1
#+end_src

对应的生成多项式比特串为：

#+begin_src text
G = 10011
#+end_src

发送端需要根据上述生成多项式，为数据比特串 D 计算 CRC 校验码，并将校验码附加在原始数据后形成最终发送帧。

* 作答要求

1. 写出生成多项式 G 的阶数。
2. 根据生成多项式的阶数，说明需要在原始数据 D 后补几个 0。
3. 写出补 0 后参与模 2 除法的被除数。
4. 使用模 2 除法计算 CRC 余数。
5. 写出最终发送的完整比特串。
6. 说明接收端如何利用同一个生成多项式判断数据传输是否出错。

* 提示

CRC 计算时，需要先在原始数据后补 r 个 0，其中 r 为生成多项式的最高阶数。

模 2 除法中的减法等价于异或运算，不产生借位。

接收端收到完整比特串后，用相同的生成多项式进行模 2 除法。如果余数为全 0，则认为该帧通过 CRC 校验；如果余数不为全 0，则认为该帧在传输过程中可能发生错误。
