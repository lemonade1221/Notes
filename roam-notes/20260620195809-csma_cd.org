#+title: CSMA/CD
* 题目：CDMA 码分多址信号分析

在一个 CDMA 系统中，有 4 个站点 A、B、C、D，它们使用长度为 8 的码片序列。各站点的码片序列如下：

| 站点 | 码片序列 |
|------+--------------------------------|
| A    | +1 +1 +1 +1 +1 +1 +1 +1 |
| B    | +1 -1 +1 -1 +1 -1 +1 -1 |
| C    | +1 +1 -1 -1 +1 +1 -1 -1 |
| D    | +1 -1 -1 +1 +1 -1 -1 +1 |

系统规定：

1. 若某站点发送数据位 1，则发送该站点本身的码片序列。
2. 若某站点发送数据位 0，则发送该站点码片序列的相反数。
3. 若某站点未发送数据，则该站点对接收信号的贡献为全 0 序列。
4. 接收端收到的信号为所有发送站点码片序列的逐位叠加。

某一时刻，接收端收到的叠加码片序列为：

#+begin_src text
R = +2  0  +2  0  +2  0  +2  0
#+end_src

请根据 CDMA 解码方法，判断 A、B、C、D 四个站点分别发送的数据位，或判断其是否未发送。

* 作答要求

1. 写出 CDMA 解码时使用的计算公式。
2. 分别计算接收序列 R 与 A、B、C、D 四个站点码片序列的规格化内积。
3. 根据计算结果判断每个站点的发送状态。
4. 写出最终结论。

* 提示

CDMA 解码时，可使用如下公式计算某一站点的解码结果：

#+begin_src text
S_i = (1 / m) * Σ(R_k * C_i,k)
#+end_src

其中：

| 符号 | 含义 |
|------+------------------------------|
| S_i  | 第 i 个站点的解码结果 |
| m    | 码片序列长度 |
| R_k  | 接收序列中的第 k 个码片 |
| C_i,k | 第 i 个站点码片序列中的第 k 个码片 |

判断规则如下：

| 计算结果 | 含义 |
|----------+----------------|
| +1       | 该站点发送数据位 1 |
| -1       | 该站点发送数据位 0 |
| 0        | 该站点未发送数据 |
