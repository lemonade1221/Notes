#+title: Org-mode

* Org-mode 是什么
Org-mode 是 Emacs 中用于笔记、任务管理、日程安排、写作和导出的纯文本标记系统。

特点：
1. 文件是普通文本，扩展名一般为 =.org=
2. 用星号表示标题层级
3. 可以同时管理笔记、TODO、表格、代码块、链接、标签和导出

* 常用语法
** 标题与层级
使用 =*= 表示标题层级：
#+begin_src org
* 一级标题
** 二级标题
*** 三级标题
#+end_src

常用结构：
#+begin_src org
* 主题
** 子主题
*** 细节
#+end_src

** 段落与换行
普通文本直接书写即可。

如果需要强制换行，可以在行尾加 =\\=：
#+begin_src org
第一行\\
第二行
#+end_src

** 列表
无序列表：
#+begin_src org
- 项目一
- 项目二
  - 子项目
#+end_src

有序列表：
#+begin_src org
1. 第一步
2. 第二步
3. 第三步
#+end_src

描述列表：
#+begin_src org
- Emacs :: 文本编辑器
- Org-mode :: 笔记和任务管理系统
#+end_src

** 文字样式
#+begin_src org
*粗体*
/斜体/
_下划线_
=代码=
~等宽文本~
+删除线+
#+end_src

效果：
- =*粗体*= 表示重点
- =/斜体/= 表示强调
- =_下划线_= 表示下划线
- ==代码== 适合命令、变量名、文件名
- =~等宽文本~= 适合代码片段
- =+删除线+= 表示废弃内容

** 链接
文件链接：
#+begin_src org
[[file:README.md][README]]
#+end_src

网页链接：
#+begin_src org
[[https://orgmode.org][Org mode 官网]]
#+end_src

Org-roam 中常见的 ID 链接：
#+begin_src org
[[id:C3D918FD-0388-4A37-A64F-87C92A4182B5][Zephyr]]
#+end_src

** 图片
#+begin_src org
[[file:images/example.png]]
#+end_src

显示内联图片：
#+begin_src emacs-lisp
M-x org-display-inline-images
#+end_src

** 引用块
#+begin_src org
#+begin_quote
这里是一段引用。
#+end_quote
#+end_src

** 代码块
基本写法：
#+begin_src org
,#+begin_src python
print("hello org")
,#+end_src
#+end_src

Shell 示例：
#+begin_src org
,#+begin_src shell
ls -la
,#+end_src
#+end_src

给代码块命名：
#+begin_src org
,#+name: example-code
,#+begin_src c
int main(void) {
    return 0;
}
,#+end_src
#+end_src

执行代码块常用快捷键：
- =C-c C-c= 在代码块中执行
- =C-c '= 打开代码块编辑缓冲区

** 表格
Org 表格用 =|= 分隔列：
#+begin_src org
| 名称 | 用途     | 快捷键  |
|------+----------+---------|
| TODO | 任务管理 | C-c C-t |
| 表格 | 结构数据 | TAB     |
#+end_src

常用表格操作：
- 在表格中按 =TAB= 跳到下一个单元格
- 按 =S-TAB= 跳到上一个单元格
- 按 =RET= 跳到下一行
- =C-c C-c= 对齐表格

** 注释
单行注释：
#+begin_src org
# 这是一行注释，导出时不会出现
#+end_src

注释块：
#+begin_src org
#+begin_comment
这里的内容不会导出。
#+end_comment
#+end_src

** 标签
标签通常写在标题末尾：
#+begin_src org
* 学习 Org-mode :emacs:notes:
#+end_src

常用操作：
- =C-c C-q= 设置当前标题标签
- =C-c \= 按标签搜索

* 快捷键操作
以下快捷键是 Emacs Org-mode 的常见默认绑定。

** 大纲导航
| 快捷键      | 作用                         |

| =TAB=       | 折叠/展开当前标题            |

| =C-c C-b=   | 跳到同级上一个标题           |
| =C-c C-u=   | 跳到上级标题                 |

** 编辑标题结构
| 快捷键          | 作用                         |
|-----------------|------------------------------|
| =M-RET=         | 新建同级标题                 |
| =M-S-RET=       | 新建 TODO 标题               |
| =M-LEFT=        | 提升当前标题层级             |
| =M-RIGHT=       | 降低当前标题层级             |
| =M-UP=          | 当前标题上移                 |
| =M-DOWN=        | 当前标题下移                 |
| =C-c *=         | 将当前行变为标题             |
| =C-c -=         | 切换列表项样式               |

** 链接与跳转
| 快捷键      | 作用                         |
|-------------+------------------------------|
| =C-c C-o=   | 打开链接                     |
| =C-c C-l=   | 插入或编辑链接               |
| =C-c &=     | 返回上一个跳转位置           |

** 时间与日程
| 快捷键        | 作用                       |
|---------------+----------------------------|
| =C-c .=       | 插入日期                   |
| =C-c !=       | 插入非激活日期             |
| =C-c C-s=     | 设置 SCHEDULED             |
| =C-c C-d=     | 设置 DEADLINE              |
| =C-c C-x C-i= | 开始计时                   |
| =C-c C-x C-o= | 停止计时                   |
| =C-c C-x C-j= | 跳到当前计时任务           |

** Agenda
| 快捷键      | 作用                         |
|-------------+------------------------------|
| =C-c a=     | 打开 Org Agenda              |
| =t=         | 在 Agenda 中切换 TODO 状态   |
| =s=         | 在 Agenda 中保存所有 Org 文件 |
| =q=         | 退出 Agenda                  |

如果 =C-c a= 没有反应，需要先配置：
#+begin_src emacs-lisp
(global-set-key (kbd "C-c a") 'org-agenda)
#+end_src

* TODO 的使用
** 基本 TODO
最简单的任务：
#+begin_src org
* TODO 学习 Org-mode
#+end_src

完成后可以改为：
#+begin_src org
* TODO 学习 Org-mode
#+end_src

常用快捷键：
- =C-c C-t= 切换 TODO 状态
- =M-S-RET= 快速创建一个 TODO 项

** 常见任务状态
可以在文件头设置任务状态：
#+begin_src org
#+TODO: TODO(t) NEXT(n) WAIT(w) | DONE(d) CANCELLED(c)
#+end_src

含义：
| 状态        | 含义               |
|-------------+--------------------|
| TODO        | 待处理             |
| NEXT        | 下一步要做         |
| WAIT        | 等待别人或外部条件 |
| DONE        | 已完成             |
| CANCELLED   | 已取消             |

竖线 =|= 左边表示未完成状态，右边表示完成状态。

** TODO 示例
#+begin_src org
* TODO 整理 Zephyr 笔记
SCHEDULED: <2026-06-20 Sat>

** NEXT 补充 west build 用法
** WAIT 等待开发板测试结果
** DONE 创建基础笔记
** CANCELLED 删除重复内容
#+end_src

** 优先级
#+begin_src org
* TODO [#A] 今天必须完成的任务
* TODO [#B] 重要但不紧急
* TODO [#C] 有时间再做
#+end_src

设置优先级：
- =C-c ,= 设置当前任务优先级
- 默认优先级一般是 =B=

** 进度统计
子任务完成比例：
#+begin_src org
* TODO 学习 Org-mode [1/3]
** DONE 学习标题
** TODO 学习 TODO
** TODO 学习 Agenda
#+end_src

百分比进度：
#+begin_src org
* TODO 学习 Org-mode [33%]
** DONE 学习标题
** TODO 学习 TODO
** TODO 学习 Agenda
#+end_src

更新统计：
- =C-c C-c= 在标题上刷新进度

** 复选框
复选框适合临时清单，不一定是正式 TODO：
#+begin_src org
* 出门前检查 [1/3]
- [X] 带电脑
- [ ] 带充电器
- [ ] 关灯
#+end_src

常用快捷键：
- =C-c C-c= 切换复选框状态
- =M-S-RET= 新建复选框项目

** SCHEDULED 与 DEADLINE
=SCHEDULED= 表示计划什么时候做：
#+begin_src org
* TODO 写实验记录
SCHEDULED: <2026-06-20 Sat>
#+end_src

=DEADLINE= 表示截止日期：
#+begin_src org
* TODO 提交报告
DEADLINE: <2026-06-25 Thu>
#+end_src

快捷键：
- =C-c C-s= 设置计划日期
- =C-c C-d= 设置截止日期

** 重复任务
每天重复：
#+begin_src org
* TODO 复习英语
SCHEDULED: <2026-06-20 Sat +1d>
#+end_src

每周重复：
#+begin_src org
* TODO 周总结
SCHEDULED: <2026-06-20 Sat +1w>
#+end_src

每月重复：
#+begin_src org
* TODO 整理账单
SCHEDULED: <2026-06-20 Sat +1m>
#+end_src

** 记录完成时间
可以配置任务完成时自动记录 CLOSED 时间：
#+begin_src emacs-lisp
(setq org-log-done 'time)
#+end_src

完成后类似：
#+begin_src org
* DONE 学习 Org-mode
CLOSED: [2026-06-20 Sat 12:00]
#+end_src

** 任务备注
可以配置切换状态时记录备注：
#+begin_src emacs-lisp
(setq org-log-done 'note)
#+end_src

这样完成任务时会要求输入说明。

* Agenda 的基本用法
Agenda 用来集中查看所有 Org 文件中的任务和日程。

** 添加 Agenda 文件
#+begin_src emacs-lisp
(setq org-agenda-files '("~/org/roam-notes"))
#+end_src

也可以指定多个目录或文件：
#+begin_src emacs-lisp
(setq org-agenda-files
      '("~/org/roam-notes"
        "~/org/tasks.org"))
#+end_src

** 常用 Agenda 视图
| 按键 | 作用               |
|------+--------------------|
| =a=  | 日程视图           |
| =t=  | 所有 TODO          |
| =m=  | 按标签匹配         |
| =s=  | 搜索关键词         |

使用流程：
1. 按 =C-c a= 打开 Agenda 菜单
2. 按 =a= 查看日程
3. 按 =t= 查看所有 TODO

* Capture 快速记录
Org Capture 用来快速捕获想法、任务和日志。

** 基本配置
#+begin_src emacs-lisp
(global-set-key (kbd "C-c c") 'org-capture)

(setq org-capture-templates
      '(("t" "Todo" entry
         (file "~/org/roam-notes/tasks.org")
         "* TODO %?\n  %U\n")
        ("n" "Note" entry
         (file "~/org/roam-notes/inbox.org")
         "* %?\n  %U\n")))
#+end_src

常用快捷键：
- =C-c c= 打开 Capture
- =C-c C-c= 保存 Capture
- =C-c C-k= 取消 Capture

* 导出
Org 文件可以导出为 HTML、PDF、Markdown 等格式。

常用入口：
#+begin_src text
C-c C-e
#+end_src

常见导出：
| 操作       | 说明        |
|------------+-------------|
| =C-c C-e h h= | 导出 HTML |
| =C-c C-e h o= | 导出并打开 HTML |
| =C-c C-e l p= | 导出 PDF |
| =C-c C-e m m= | 导出 Markdown |

* 文件头常用设置
#+begin_src org
#+title: 笔记标题
#+author: 作者
#+date: 2026-06-20
#+startup: overview
#+tags: emacs org notes
#+TODO: TODO(t) NEXT(n) WAIT(w) | DONE(d) CANCELLED(c)
#+end_src

常用 =#+startup= 选项：
| 设置                  | 作用               |
|-----------------------+--------------------|
| =#+startup: overview= | 默认只显示顶层标题 |
| =#+startup: content=  | 显示所有标题       |
| =#+startup: showall=  | 展开全部内容       |
| =#+startup: inlineimages= | 默认显示图片   |

* 实用工作流
** 笔记工作流
1. 用标题组织知识结构
2. 用链接连接相关笔记
3. 用标签做主题分类
4. 用代码块保存可执行示例
5. 用表格整理对比信息

** 任务工作流
1. 用 =TODO= 记录待办
2. 用 =NEXT= 标记下一步动作
3. 用 =SCHEDULED= 安排开始时间
4. 用 =DEADLINE= 标记截止时间
5. 用 Agenda 每天统一查看

** 推荐状态流转
#+begin_src org
TODO -> NEXT -> DONE
TODO -> WAIT -> NEXT -> DONE
TODO -> CANCELLED
#+end_src

* 常见问题
** 表格没有对齐
在表格中按：
#+begin_src text
C-c C-c
#+end_src

** 链接打不开
检查链接格式：
#+begin_src org
[[链接地址][显示文字]]
#+end_src

打开链接使用：
#+begin_src text
C-c C-o
#+end_src

** TODO 不出现在 Agenda 中
检查：
1. 文件是否在 =org-agenda-files= 中
2. 任务是否使用了正确的 TODO 关键字
3. 是否保存了文件

** 代码块不能执行
需要启用对应语言支持，例如：
#+begin_src emacs-lisp
(org-babel-do-load-languages
 'org-babel-load-languages
 '((python . t)
   (shell . t)
   (C . t)))
#+end_src
